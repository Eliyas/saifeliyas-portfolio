### Prerequisites

**Node version 14.x**

### Cloning the repository

```shell
git clone https://github.com/Eliyas/saifeliyas-portfolio.git
```

### Install packages

```shell
npm i
```

### Setup .env.local file


```js
BASE_URL=
```

### Start the app

```shell
npm run dev
```

## Available commands

Running commands with npm `npm run dev`

| command         | description                              |
| :-------------- | :--------------------------------------- |
| `dev`           | Starts a development instance of the app |
